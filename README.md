# metory-dapp

METORY - The real-world feasibility of a blockchain-based dynamic consent platform 



### Installation 

환경 

```
nodejs - v14.18.0
npm - v6.14.15
```



라이브러리 패키지 설치

```
npm install 
```



설정파일 

```bash
#Wallt 정보 
/controller/features/fabric/_idwallet

#블록체인 접속 기본정보
/controller/features/fabric/config.json

#기관별 profile 업로드 폴더 
/uploads/기관명/profile.json 
```
